    How Tracker Items Work

    What are tracker items?

    Tracker items are tasks that students must respond to (Done / Not Done) at checkout. They help track daily habits, assignments, and goals across all students.

    Recurrence

    • Daily — appears every day. Resets each day.
    • Weekly — appears once per week (Monday–Sunday). Once responded, it won't appear again until next week.
    • Monthly — appears once per month. Once responded, it won't appear again until next month.
    • One-time — appears until the student marks it done.

    Start Date & End Date

    These define the active window for the item. Both are optional.

    • Neither set — the item is always active (no date restrictions).
    • Start date only — the item becomes active on that date and continues indefinitely.
    • End date only — the item is active immediately and stops appearing after the end date.
    • Both set — the item is only active within that date range.

    Progress & Completion

    Progress is calculated based on expected completions, not just responses. If a daily item is active for 5 days and a student only marks it done on 3, their completion is 60% — the 2 missed days count against them.

    Example

    A "Homework Check" item set to Daily with start date Apr 1 and end date Apr 30 will appear every day in April. A student who checks in on 20 of those days and marks it done each time has 20/30 = 67% completion.
